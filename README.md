Hotel Room Management System
 📌 Description
This is a Java GUI based Hotel Room  management system.
1.	Customer Features
•	Register account
•	Login / Logout
•	View available rooms
•	Book room
•	Cancel booking
•	View booking history
•	Make payment

2.	Staff Features

•	Login / Logout
•	View bookings
•	Update booking
•	Delete booking
•	Manage rooms
•	View payments

3.	Admin Features

•	Login / Logout
•	Manage customers
•	Manage rooms (Add / Update / Delete)
•	Manage staff
•	Generate reports

 Technologies

•	Java
•	Swing
•	MySQL

 How to Run

Open project in IDE
Run Main.java
